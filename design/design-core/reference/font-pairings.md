# Font Pairings

One line = one decision. Match on mood and context, not habit.
All fonts available on Google Fonts unless noted. Import example: `@import url('https://fonts.googleapis.com/css2?family=Font+Name:wght@400;700&display=swap');`

## Editorial / literary / blog
- Libre Bodoni + Source Serif 4 → `family=Libre+Bodoni:wght@400;700|Source+Serif+4:wght@400;600` *(classic editorial)*
- EB Garamond + Libre Baskerville → `family=EB+Garamond:wght@400;600;700|Libre+Baskerville:wght@400;700` *(academic, archival)*
- Abril Fatface + Merriweather → `family=Abril+Fatface|Merriweather:wght@400;700` *(vintage editorial, strong hierarchy)*
- Literata + Charter *(literary, book-like)* — Charter via system or self-host

## Tech / developer
- JetBrains Mono + Fira Sans → `family=JetBrains+Mono:wght@400;700|Fira+Sans:wght@400;500;700` *(code-heavy, terminal)*
- Figtree + Noto Sans → `family=Figtree:wght@400;600;700|Noto+Sans:wght@400;500;700` *(clean, scientific, readable)*
- Exo + Source Code Pro → `family=Exo:wght@400;600;700|Source+Code+Pro:wght@400;600` *(science, data platform)*

## SaaS / product / startup
- Outfit + Work Sans → `family=Outfit:wght@400;600;700|Work+Sans:wght@400;500;700` *(modern, geometric)*
- Satoshi + General Sans *(premium, contemporary)* — via Fontshare (free)
- Clash Display + Satoshi *(bold startup)* — via Fontshare
- Lexend + Source Sans 3 → `family=Lexend:wght@400;500;700|Source+Sans+3:wght@400;600;700` *(accessible, trustworthy)*
- Albert Sans + Geist Sans *(modern product)* — Geist via Vercel
