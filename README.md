# my-skills

A curated set of AI skills for design, writing, and research — with more to come.

LLMs are already capable. A skill's job is to focus that capability, not replace it — specialized enough to be precise, minimal enough to be cheap. Every token in a skill is a token that could have gone to your actual task, so each one has to earn its place. No filler, no redundancy — just the signal that unlocks the right behavior across any context.

## Install

```bash
npx skills add github:tncrd/skills
```

Or clone and copy manually:

```bash
git clone https://github.com/tncrd/skills
cp -r my-skills/skills/ .claude/skills/
```

## Skills

| Group | Pipeline | Focus |
|-------|----------|-------|
| design | `design-shape` → `design` → `design-audit` → `design-polish` | UI, typography, color, motion |
| research | `research` → `research-ideation` → `research-review` | Literature search, paper notes |
