# Skills

Two skill groups, each a pipeline. Install all or by group.

```bash
npx skills add github:tncrd/skills
npx skills add github:tncrd/skills --group design
npx skills add github:tncrd/skills --group research
```

## Design

`design-shape` → `design` → `design-audit` → `design-polish`

Shape discovers intent, design builds with on-demand references, audit scores /20 with prioritized issues, polish runs the pre-ship checklist.

→ See [design/README.md](design/README.md)

## Research

`research` → `research-ideation` → `research-review`

Research handles literature search and paper reading, ideation generates hypotheses from frameworks, review maps an entire field and builds a bibliography.

→ See [research/README.md](research/README.md)
